In this project, we will be using TransE with Margin Ranking Loss. We will also implement a negative sampler that will be used to differentiate positives from negatives.

Negative Sampler

For each triple {s,r,o}\{s,r,o\}{s,r,o} in the dataset. We will be corrupting the triple by either replace the head sss or object ooo to treat it as a negative sample. Note that a corrupted triple may still be a positive in the original graph.
